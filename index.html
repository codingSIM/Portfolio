Projects

Weekado

Weekado is a productivity application based on the book "Indistractable" by Nir Eyal. I have used LAMP stack to develop the application. Additionally cron and bash have been used for parts of the functionality. Weekado has 3 main features: To-do list, Pomodoro and Weekly Habit Tracker.

Chalk

Chalk is a 2D platformer game created using p5.js. In the game you play as a piece of chalk. The mechanic of the game revolves around life draining linked to movement (a.k.a. "Chalk" mechanics). The Hall of Fame version contains only one level, however a demo is under development.

Hall of Fame

Portativ 2

Portativ is a music tracker created using JavaScript (+p5.sound). It contains functionality such as: playing/pausing, recording, saving progress, loading progress and other extra features. This project also inludes keyboard animations and changing octaves. Check it out!

Github

Robospar

Robospar is a 2 player, simple grid-based game with 3 possible moves. The moves are denoted as: F (move forward), R (rotate) and Z (zap). Both players have to enter their string of moves at the same time, it is not turn based. The objective is to either reach your base first or zap your oponent.
